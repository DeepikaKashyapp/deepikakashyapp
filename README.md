## 🌸 Featured Projects

### 🛍️ Multimodal Product Inconsistency Detector
*Jun 2026 — Personal Project*

A catalog-auditing cutie that uses OpenAI **CLIP's Dual-Encoder (ViT-B/32)** to sniff out semantic mismatches between e-commerce product images and titles 🕵️‍♀️

- 💫 Projects images & text into a shared 512-d latent space, then computes cosine similarity between L2-normalized embeddings
- 🎯 F1-optimized dynamic thresholding (100-candidate grid search) to auto-flag inconsistent listings
- 📈 **77.8% accuracy** · **ROC-AUC 0.76** · **80% precision** — tuning the threshold from 0.25 → 0.21 pushed mismatch recall to **100%** with zero retraining
- 🚀 Deployed via Streamlit for real-time single-product inference + a standalone evaluation pipeline (confusion matrix, ROC/PR curves, per-product dashboards)

**Tech:** Python · CLIP · HuggingFace · PyTorch · Streamlit · Scikit-learn

### ♻️ Zero-Waste Vision (AI Waste Classifier)
*May 2026 — Personal Project*

An eco-friendly little system that sorts waste in real time using **YOLOv8n + MobileNetV2** 🌍💚

- ⚡ Localized inference at **162.1 ms/frame (~6 FPS)** on CPU — lightweight and snappy!
- 🗑️ Custom dictionary-mapping algorithm linking 80+ object classes to disposal protocols (Plastic, E-Waste, Organic) + estimated carbon savings
- 🔗 ML models served via a Flask REST API to a React frontend for a smooth real-time experience

**Tech:** Python · YOLOv8 · MobileNetV2 · Flask · React
