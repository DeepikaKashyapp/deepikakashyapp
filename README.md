<h3 align="center">✦ things i've built</h3>

<table align="center" width="100%" cellpadding="14">
<tr>
<td width="50%" valign="top">

<div>

**multimodal product inconsistency detector**
<sub>personal project · jun 2026</sub>

A catalog-auditing tool that uses CLIP's dual-encoder to catch when a product photo and its title don't actually agree with each other — projects both into a shared 512-d space and flags the gap.

<sub>77.8% accuracy · 0.76 ROC-AUC · tuned to 100% mismatch recall</sub>

<sub>`Python` `CLIP` `PyTorch` `Streamlit`</sub>

</div>

</td>
<td width="50%" valign="top">

<div>

**zero-waste vision**
<sub>personal project · may 2026</sub>

An AI waste classifier (YOLOv8n + MobileNetV2) that looks at trash and tells you where it actually belongs — fast enough to run live on a CPU, ~6 frames a second.

<sub>80+ object classes mapped to disposal protocols + estimated carbon savings</sub>

<sub>`Python` `YOLOv8` `Flask` `React`</sub>

</div>

</td>

**echochat**
<sub>team project · jan 2026</sub>

<sub>custom priority triage + packet-dropping protocol for zero-latency SOS routing</sub>

<sub>`Android` `Bluetooth LE` `Wi-Fi Direct`</sub>
